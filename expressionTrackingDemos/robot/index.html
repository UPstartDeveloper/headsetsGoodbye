Roberto: The Empathetic Robot

via GIPHY

How to Play:

  1. You can control Roberto's facial expressions via the control panel or your own facial expressions in the camera.
      Supported Expressions:
    • Anger
    • Surprise
    • Sadness
  2. Roberto loops in one of several states such as "Walking", "Sitting", "Standing", etc., which are shown on the control panel in the top right of the next page.
  3. Roberto can also perform one-off actions such as "Jump", "Wave", "ThumbsUp", etc. that are also shown on the control panel.
  4. To begin, select the green button below that says "Start Expression Tracking"!
